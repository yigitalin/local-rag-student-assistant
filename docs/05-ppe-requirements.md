# PPE Requirements for Gas Field Operations

## Purpose
Mandatory Personal Protective Equipment requirements for all gas field operations.

## Safety Warning
- Entry to ANY gas installation site without correct PPE is PROHIBITED.
- PPE must be inspected before each use – do NOT use damaged equipment.

## Standard PPE (All Site Activities)
| Item | Standard | Inspection Frequency |
|------|----------|---------------------|
| Hard hat | EN 397 | Before each use |
| Safety glasses | EN 166 | Before each use |
| Flame-resistant coveralls | EN 11612 | Before each use |
| Steel-toe safety boots | EN 20345 S3 | Before each use |
| Cut-resistant gloves | EN 388 Level 3+ | Before each use |
| Hi-visibility vest | EN 20471 Class 2+ | Before each use |

## Additional PPE (Hazardous Zone Work)
| Item | Standard | When Required |
|------|----------|---------------|
| Personal gas monitor | EN 60079-29-1 | All gas-hazardous areas |
| SCBA (Self-Contained Breathing Apparatus) | EN 137 | Confined space / high LEL |
| Chemical-resistant gloves | EN 374 | Chemical handling |
| Face shield | EN 166 B grade | Pressure testing |
| Hearing protection | EN 352 | Noise >85 dB(A) |
| Fall arrest harness | EN 361 | Working at height >2m |

## Gas Monitor Requirements
- Must be bump-tested before each shift.
- Full calibration every 30 days minimum.
- Alarms set to:
  - Low: 10% LEL
  - High: 20% LEL
  - TWA: 1 ppm H₂S
  - STEL: 5 ppm H₂S

## Inspection and Replacement
- Inspect all PPE before each use.
- Replace hard hats every 3 years or after any impact.
- Replace harnesses every 5 years or after any arrest event.
- Flame-resistant clothing must be replaced if FR treatment is compromised.
