# Domain Candidates

## Objective

Select a technical domain that is as close as possible to Microsoft's original Gas Field Support Agent.

The selected domain should require minimal architectural and document changes while preserving the original Local RAG design.

## Final Decision

### Selected Domain

**Oil Refinery Equipment Support Agent**

### Reason

The selected domain provides the highest compatibility with the original Microsoft Gas Field Support Agent while requiring minimal modifications to the project architecture, Markdown document structure, source code, and RAG workflow.

The equipment types, maintenance procedures, safety guidelines, troubleshooting documents, and operational workflows closely match those used in the Microsoft reference implementation.

This allows the project to remain an adaptation of the original reference application rather than a redesign.
