# Pre-Inspection Checklist

## Purpose
Standard pre-inspection checklist to be completed before commencing any gas infrastructure field inspection.

## Safety Warning
- Do NOT begin inspection until all checklist items are confirmed.
- If any item cannot be confirmed, notify supervisor before proceeding.

## Checklist

### Personal Safety
- [ ] PPE inspection: hard hat, safety glasses, flame-resistant clothing, steel-toe boots, gloves
- [ ] Gas monitor calibrated and functional (calibration date within 30 days)
- [ ] Communication radio tested – confirm contact with control room
- [ ] First aid kit available on-site
- [ ] Buddy system in place (lone working prohibited in hazardous zones)
- [ ] Risk assessment reviewed and signed (RA form)

### Site Preparation
- [ ] Site access confirmed and secured
- [ ] Weather conditions assessed (wind direction, precipitation)
- [ ] Exclusion zone established if required
- [ ] Ignition source survey completed
- [ ] Emergency escape routes identified and briefed
- [ ] Nearest emergency services location confirmed

### Equipment
- [ ] Pressure gauges calibrated (within 12 months)
- [ ] Leak detection equipment operational
- [ ] Isolation tools present and correct
- [ ] Torque wrench calibrated (within 6 months)
- [ ] Spare gaskets and seals available (correct spec)
- [ ] Camera/recording device for evidence capture

### Documentation
- [ ] Work permit obtained and valid
- [ ] Method statement reviewed
- [ ] Previous inspection records available
- [ ] Asset register reference confirmed
- [ ] Emergency contact numbers available
